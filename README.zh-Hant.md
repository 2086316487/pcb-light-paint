# PCB燈光畫

PCB燈光畫是用於 JLCEDA / EasyEDA 專業版 PCB 場景的外掛。  
它提供圖片到 PCB 分層素材的完整流程，並支援在 PCB 文件內一鍵生成圖元。

## 主要功能

- 上傳圖片並進行原圖編輯（畫筆/套索/填充）
- 固定調色盤量化簡化
- 生成遮罩圖、圖層圖與實物預覽圖
- 單圖保存與 ZIP 匯出
- 一鍵生成 PCB 畫圖元

## 快速開始

1. 安裝 `build/dist/pcb-light-paint_v0.1.6.eext`
2. 開啟 PCB 文件
3. 進入 `PCB燈光畫 -> 打開工作台`
4. 上傳與處理圖片
5. 點擊 `一键生成PCB画` 或匯出檔案
