## 概要

- JLCEDA / EasyEDA Pro の PCB ワークスペース向けプラグイン
- 画像編集、量子化、マスク/レイヤー生成、ZIP 出力をサポート
- PCB ドキュメント内で「一键生成PCB画」により図元生成が可能
