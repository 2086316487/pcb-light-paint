# PCB Light Paint

Эта страница временно содержит краткое описание.  
Подробности: [README.md](./README.md) и [README.en.md](./README.en.md).

## Описание

- Плагин для PCB-сцены в JLCEDA / EasyEDA Pro
- Поддерживает обработку изображения, квантование, генерацию масок/слоев и ZIP-экспорт
- Поддерживает создание PCB-примитивов через кнопку `一键生成PCB画`

## Версия

- `0.1.6`

## Текущий статус
